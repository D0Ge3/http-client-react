# Patient portal
React HTTP client
(Тестовое задание)

## Stack

- React
- Redux + redux-thunk
- Redux Toolkit
- React-router
- Sass modules

## Structure

- `/src/components` - components folder
- `/src/common` - reusable components
- `/src/redux` - redux store and etc, with business logic

## Commands
### yarn
```bash
# Install modules
yarn install 
# Run app in dev mode
yarn start
# Create production build
yarn build
```

### npm
```bash 
# Install modules
npm install 
# Run app in dev mode
npm start
# Create production build
npm build
```
